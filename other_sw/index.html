CD Reconstruction from several damaged copies

Imagine you have two or three identical CDs, but all scratched somewhere. How to make one completely good?

Let's merge good bits and pieces from different CDs. To be simple, this program works only with CD images. I used CloneCD 4 to read/write CD images. If you don't have CloneCD, you can simply improve this program to work with files from your program.
