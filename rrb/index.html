REALTIME RADIOSITY BUGS

Bugs sneak in dark corners, hunt them!
(In fact, they are space invaders!)

Realtime Radiosity Bugs present experimental concept of game based on realtime updated global illumination, realtime radiosity.
The demo contains realtime radiosity with true soft shadows and color bleeding, all coming from dynamic light source you can control.
To run it, you need OpenGL 2.0 capable graphics card, eg. GeForce 6100 or higher, Radeon 9500 or higher, with recent drivers.

Nothing in the demo is precalculated, you can load your own .3ds scene.
